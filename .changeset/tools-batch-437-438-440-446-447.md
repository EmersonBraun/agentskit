feat(tools): five new integrations:

- `twilioSmsTool` (#438) — Twilio SMS via the REST Messages endpoint with E.164 validation.
- `pagerduty` (#440) — Events API v2 (`trigger` / `acknowledge` / `resolve`) + optional REST `oncall` lookup.
- `stripeWebhookTool` (#437) — verifies the `Stripe-Signature` header (HMAC-SHA256, replay-tolerant) and returns the parsed event; throws on any mismatch.
- `cloudflareR2` (#446) — R2 storage via the S3-compatible protocol; `r2_get` / `put` / `list` / `delete` / `signed_url`. `@aws-sdk/client-s3` is an optional peer dependency.
- `postgresWithRoles` (#447) — read/write-split helper. Two role-bound clients map to two distinct tools (`postgres_read` / `postgres_write`), enforcing least-privilege at the database level instead of in the prompt.

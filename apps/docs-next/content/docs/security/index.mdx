## Primitives

- **PII redaction** — `createPIIRedactor` + `DEFAULT_PII_RULES`. [Recipe](/docs/recipes/pii-redaction).
- **Prompt injection detector** — heuristics + pluggable model classifier. [Recipe](/docs/recipes/prompt-injection).
- **Rate limiting** — token-bucket by user / IP / key. [Recipe](/docs/recipes/rate-limiting).
- **Signed audit log** — hash-chain + HMAC. [Recipe](/docs/recipes/audit-log).
- **Mandatory sandbox** — allow / deny / require / validators across every tool. [Recipe](/docs/recipes/mandatory-sandbox).
- **Human-in-the-loop approvals** — pause / resume / approve with persisted state. [Recipe](/docs/recipes/hitl-approvals).

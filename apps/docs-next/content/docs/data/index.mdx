Three substrates every agent touches:

- **[Memory](/docs/data/memory)** — chat history + vector stores + higher-order wrappers (hierarchical, encrypted, graph, personalization).
- **[RAG](/docs/data/rag)** — chunk, embed, retrieve, rerank, hybrid. Six document loaders ship built in.
- **[Providers](/docs/data/providers)** — LLM chat + embedder adapters (20+), plus higher-order router / ensemble / fallback.

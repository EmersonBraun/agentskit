Once you have built a real agent, these six contracts explain why the ecosystem composes so cleanly. Every package, every provider, every memory backend, every tool, and every UI binding plugs into one of them.

If you have not done that yet, start with [Build your first agent](/docs/get-started/getting-started/build-your-first-agent) and come back here after you have seen the stack in motion.

## The six contracts

| Contract | What it abstracts | Page |
|---|---|---|
| **Adapter** | LLM providers | [adapter](./adapter) |
| **Tool** | callable functions | [tool](./tool) |
| **Skill** | personas + prompts | [skill](./skill) |
| **Memory** | conversation + vector state | [memory](./memory) |
| **Retriever** | context retrieval | [retriever](./retriever) |
| **Runtime** | the orchestration loop | [runtime](./runtime) |

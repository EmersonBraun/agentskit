# Empieza aquí (~60 segundos)

1. **[Inicio rápido](./quick-start)** — copia y pega un chat con streaming; cambia el adaptador en una línea.
2. **[Descripción general de paquetes](../packages/overview)** — los catorce paquetes `@agentskit/*` y enlaces a cada guía.
3. **Abre la guía de lo que despliegas** — React, Ink, runtime, RAG, etc., desde esa tabla.

**Firmas y tipos:** [API generada (TypeDoc)](pathname:///agentskit/api-reference/) (ejecuta `pnpm --filter @agentskit/docs docs:api` una vez si usas `docusaurus start` en local).

**Chuleta de instalación:** [Instalación](./installation).

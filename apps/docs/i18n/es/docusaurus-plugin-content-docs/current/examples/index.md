# Ejemplos

Demos interactivas que muestran qué puedes construir con AgentsKit. Cada ejemplo es un chat — porque eso es lo que hace AgentsKit.

## Patrones de chat

| Ejemplo | Qué muestra |
|---------|---------------|
| [Chat básico](./basic-chat) | Conversación en streaming, botón de parar, manejo del teclado |
| [Uso de herramientas](./tool-use) | Llamadas a funciones — clima, búsqueda, con tarjetas de resultado expandibles |
| [Multi-modelo](./multi-model) | Comparación lado a lado Claude frente a GPT |
| [Asistente de código](./code-assistant) | Código en streaming con resaltado de sintaxis |
| [Atención al cliente](./support-bot) | Respuestas rápidas, indicador de escritura, flujos de escalado |
| [Chat RAG](./rag-chat) | Contexto de documentos, citas, referencias a fuentes |
| [Acciones de agente](./agent-actions) | La IA genera UI interactiva en vivo (seguimiento de tareas, panel) |
| [Chat Markdown](./markdown-chat) | Respuestas enriquecidas — tablas, listas, bloques de código, citas |
